# Student Management System

A console-based **Student Management System** developed in **C++** as a second-semester **Object-Oriented Programming (OOP)** project at FAST NUCES.

The project demonstrates the practical implementation of core OOP concepts including encapsulation, inheritance, polymorphism, abstraction, and file handling while simulating a simple academic management system with multiple user roles.

---

## 📖 Overview

The Student Management System provides separate portals for different types of users:

- **Admin**
- **Teacher**
- **Student**

Each role has its own permissions and functionalities, allowing users to interact with the system according to their responsibilities.

The project stores and retrieves data using file handling, enabling basic persistence between program executions.

---

## ✨ Features

### 👨‍💼 Admin

- Secure login
- Manage student records
- Manage teacher records
- View system data
- Update and delete records

### 👨‍🏫 Teacher

- Secure login
- View assigned students
- Manage student marks
- Update academic information

### 👨‍🎓 Student

- Secure login
- View personal profile
- View academic information
- Check marks and results


Using g++:


Linux/macOS
